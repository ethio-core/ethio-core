# User Guide

## Introduction

Welcome to Ethio-Core! This guide will help you understand how to use the platform for digital identity verification, card management, and financial transactions.

## Getting Started

### Creating an Account

1. Navigate to the registration page
2. Enter your email address and phone number
3. Create a secure password (minimum 8 characters with mixed case, numbers, and symbols)
4. Verify your email address by clicking the link sent to your inbox
5. Complete phone verification via SMS code

### Completing KYC Verification

To access full platform features, you must complete Know Your Customer (KYC) verification.

#### Level 1 - Basic Verification
- Email verification
- Phone verification
- Daily transaction limit: 5,000 ETB

#### Level 2 - Standard Verification
- National ID document scan
- Selfie verification
- Daily transaction limit: 50,000 ETB

#### Level 3 - Full Verification
- National ID + Fayda verification
- Biometric enrollment (face)
- Liveness detection
- Daily transaction limit: 500,000 ETB

### Document Requirements

**National ID Card:**
- Clear, high-resolution image
- All four corners visible
- No glare or reflections
- Both front and back required

**Selfie Verification:**
- Well-lit environment
- Face clearly visible
- No sunglasses or hats
- Match the photo on your ID

## Digital Wallet

### Viewing Your Balance

Your wallet dashboard displays:
- Current balance
- Available balance (after pending transactions)
- Recent transactions
- Monthly spending summary

### Adding Funds

**Bank Transfer:**
1. Go to "Add Money" in the dashboard
2. Select "Bank Transfer"
3. Copy the provided reference number
4. Transfer from your bank using the reference
5. Funds appear within 1-2 business hours

**Mobile Money:**
1. Go to "Add Money"
2. Select your mobile money provider
3. Enter the amount
4. Confirm via mobile money PIN
5. Funds appear instantly

### Withdrawing Funds

1. Go to "Withdraw" in the dashboard
2. Select destination (bank account or mobile money)
3. Enter the amount
4. Confirm with your PIN/biometric
5. Processing time: 1-3 business days for banks, instant for mobile money

## Card Management

### Virtual Card

A virtual card is issued automatically upon account verification.

**Features:**
- Instant issuance
- Use for online purchases
- Dynamic CVV for extra security
- Real-time transaction notifications

**Viewing Card Details:**
1. Go to "Cards" section
2. Click on your virtual card
3. Verify with biometric/PIN to view full details
4. Card number, expiry, and CVV displayed

### Physical Card (Coming Soon)

Request a physical card:
1. Go to "Cards" section
2. Click "Request Physical Card"
3. Enter delivery address
4. Pay issuance fee
5. Card delivered within 7-10 business days

### Card Controls

**Freeze/Unfreeze:**
- Temporarily disable your card
- Useful when card is misplaced
- Instant activation when found

**Transaction Limits:**
- Set daily spending limits
- Set per-transaction limits
- Category-based restrictions (e.g., no gambling)

**Notifications:**
- Real-time transaction alerts
- Suspicious activity warnings
- Balance threshold alerts

## Making Transactions

### Online Purchases

1. At checkout, enter your card details
2. For Dynamic CVV cards, use the CVV shown in the app (refreshes every 5 minutes)
3. Complete 3D Secure verification if prompted
4. Receive instant transaction notification

### QR Code Payments

1. Open the app and tap "Pay"
2. Scan the merchant's QR code
3. Verify the amount and merchant name
4. Confirm with biometric/PIN
5. Transaction completes instantly

### Person-to-Person Transfers

1. Go to "Send Money"
2. Enter recipient's phone number or email
3. Enter the amount
4. Add an optional note
5. Confirm with biometric/PIN
6. Recipient receives funds instantly

### Offline Transactions

For areas with limited connectivity:

1. Enable "Offline Mode" in settings
2. Ensure you have sufficient offline balance
3. Make transactions as usual
4. Transactions queue and process when online
5. Maximum offline transaction: 2,000 ETB
6. Maximum offline balance: 10,000 ETB

## Security Features

### Two-Factor Authentication (2FA)

Enable 2FA for extra security:
1. Go to Settings > Security
2. Enable "Two-Factor Authentication"
3. Choose method: SMS, Email, or Authenticator App
4. Complete setup verification

### Biometric Authentication

**Face Recognition:**
- Used for login and transaction approval
- Requires initial enrollment during KYC
- Liveness detection prevents photo attacks

**PIN:**
- 6-digit numeric PIN
- Used as backup when biometric fails
- Change regularly for security

### Transaction Verification

High-value transactions require additional verification:
- Transactions over 10,000 ETB: PIN + Biometric
- New recipients: Additional verification code
- Suspicious patterns: Manual review

### Security Alerts

You'll be notified about:
- Login from new device
- Failed login attempts
- Large transactions
- Card used in new location
- Profile changes

## Troubleshooting

### Common Issues

**Can't Log In:**
- Verify email/phone is correct
- Check Caps Lock for password
- Try "Forgot Password" if unsure
- Contact support after 5 failed attempts

**Transaction Declined:**
- Check available balance
- Verify daily limit not exceeded
- Ensure card is not frozen
- Check if merchant category is blocked

**KYC Verification Failed:**
- Ensure documents are clear and readable
- Verify document is not expired
- Retake selfie in better lighting
- Contact support for manual review

**Biometric Not Working:**
- Clean your camera lens
- Ensure good lighting
- Position face within the frame
- Use PIN as alternative
